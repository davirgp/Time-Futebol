Projeto desenvolvido em Java com o objetivo de simular o gerenciamento de clubes de futebol.

No sistema, o usuário poderá criar um clube do zero e transformá-lo em uma grande time por meio da administração esportiva e financeira da equipe. 
Será possível contratar jogadores, negociar transferências, definir escalações, administrar estádios, contratar técnicos, desenvolver atletas e disputar competições.


Tecnologias utilizadas:

- Java
- Maven
- Swing
- CSV para banco de dados
- Git


Banco de Dados:

O projeto possui um sistema próprio de banco de dados contendo centenas de jogadores reais.

Cada jogador possui:

- Nome
- Idade
- Posicao
- Overall
- Salário
- Valor de tranferência
- Estatísticas
- Atributos técnicos e físicos
